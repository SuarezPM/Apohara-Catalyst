# Apohara Agent Mistakes Log

> Auto-populable from ledger events `repeated_rejection` and manual entries.
> Format: most-recent at the top.
>
> CLI tool: `apohara incident extract <run-id>` (Stage 7+) reads from ledger
> and proposes an entry. Manual entries also welcome.

## Template

```
## YYYY-MM-DD — <Short title>

**Run:** r-xxx
**Provider:** claude / codex / opencode
**What happened:** ...
**Root cause:** ...
**Lesson:** ...
**Fix:** ...

> User feedback verbatim: "..."
```
